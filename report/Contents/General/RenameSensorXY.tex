\section*{Einleitung}
Dieses Kapitel beschreibt den HC-SR04 Ultraschallsensor im Detail, einschließlich seiner Hardware-Spezifikationen, der verwendeten Software-Bibliotheken sowie praktischer Anwendungs- und Codebeispiele.
\Mynote{\cite{berrybase, sparkfun, arduino_newping}}

\section{Allgemein}
Der HC-SR04 ist ein Ultraschallsensor zur  Entfernungsmessung. Der Sensor sendet  Schallwellen aus und misst die Dauer, bis das Echo von einem reflektierenden Objekt zurückkommt. Anhand dieser Dauer lässt sich die exakte Distanz zum Objekt präzise berechnen. Typische Einsatzgebiete in der Praxis umfassen die Robotik (Hinderniserkennung), Automatisierungstechnik, Sicherheitssysteme sowie Füllstandsmessungen und Abstandswarner (z.B. Einparkhilfen).

\section{HC-SR04}
Der HC-SR04-Sensor besteht grundlegend aus drei Hauptkomponenten: einem Ultraschallsender (Transmitter), einem Ultraschallempfänger (Receiver) und einer integrierten Steuerschaltung. Er verfügt über vier Pins zur Verbindung mit der Hardware: \texttt{VCC} (Stromversorgung), \texttt{TRIG} (Triggereingang), \texttt{ECHO} (Ausgang für das Messergebnis) und \texttt{GND} (Masse). Um den Sensor mit einem Entwicklungsboard anzusteuern, wird ein mindestens 10 Mikrosekunden langer HIGH-Impuls an den \texttt{TRIG}-Pin gesendet. Daraufhin sendet der Sensor vollautomatisch acht Ultraschallimpulse mit einer Frequenz von 40 kHz aus und setzt parallel den \texttt{ECHO}-Pin auf HIGH. Sobald das Ultraschallsignal reflektiert wird und zum Empfänger zurückkehrt, fällt der \texttt{ECHO}-Pin wieder auf LOW. Die Dauer, die der \texttt{ECHO}-Pin auf HIGH stand, entspricht der Schalllaufzeit.

	#include <NewPing.h>

	#define ECHO_PIN     11  // Arduino Pin verbunden mit ECHO
	#define MAX_DISTANCE 400 // Maximale Reichweite in cm

	void setup() {
		Serial.begin(9600);
	}
	
	void loop() {
		delay(50); // Empfohlene Mindestwartezeit zwischen Pings (ca. 20 pings/sec)
		unsigned int uS = sonar.ping(); // Pingen und Zeit in Mikrosekunden erhalten

		Serial.print(uS / 58); // Formel: Mikrosekunden / 58 = Zentimeter
		Serial.println(" cm");
	}
\end{verbatim}

\section{Kalibrierung}
Die Messung der Ultraschall-Laufzeit basiert physikalisch auf der Ausbreitung von Schall in der Luft (Schallgeschwindigkeit), welche bei ca. 20 °C exakt 343 m/s beträgt. 
Die standardisierte Berechnungsformel lautet: 
\[ \text{Distanz (cm)} = \frac{\text{Laufzeit (µs)}}{58} \]
oder als Rohberechnung:
\[ \text{Distanz} = \frac{\text{Laufzeit} \times \text{Schallgeschwindigkeit}}{2} \]
(Dividiert durch 2, da der Schall die Strecke zum Objekt und wieder zurück zurücklegen muss). 
Da die Schallgeschwindigkeit temperaturabhängig ist, kann für hochpräzise Kalibrierungen ein zusätzlicher Temperatursensor in die Schaltung integriert werden, um den Teilerfaktor in Echtzeit anzupassen.

	void setup() {
		pinMode(trig, OUTPUT);
		pinMode(echo, INPUT);
		Serial.begin(9600);
	}
	
	void loop() {
		// Trigger-Impuls erzeugen (10 Mikrosekunden)
		digitalWrite(trig, LOW);
		delayMicroseconds(2);
		digitalWrite(trig, HIGH);
		delayMicroseconds(10); 
		digitalWrite(trig, LOW);
		
		// Zeit messen, bis Echo-Pin auf LOW fällt
		long duration = pulseIn(echo, HIGH);
		float distance = duration / 58.0; 
		
		Serial.print("Distanz: ");
		Serial.print(distance);
		Serial.println(" cm");
		delay(100);
	}
\end{verbatim}

\section{Allgemeines}
Sensoren zur Abstandsmessung sind wichtige  Komponenten in der modernen Automatisierungstechnik, Robotik und Fahrzeugtechnik \cite{Fraden:2016}. Sie ermöglichen es Systemen, eine Umgebung räumlich zu erfassen. Grundsätzlich lassen sich diese Sensoren in verschiedene Kategorien einteilen:
\begin{itemize}
	\item \textbf{Infrarotsensoren:} Nutzen das Reflexionsverhalten von Licht oder messen die Lichtlaufzeit (Time-of-Flight, ToF). Sie sind extrem schnell und präzise, jedoch stark abhängig von den Oberflächeneigenschaften (Farbe, Reflexionsgrad) des Zielobjekts und anfällig für Fremdlichteinstrahlung \cite{Fraden:2016}.
	\item \textbf{Radarsensoren:} Arbeiten mit elektromagnetischen Wellen im Radio- oder Mikrowellenbereich. Sie sind unempfindlich gegenüber Schmutz, Nebel oder Lichtverhältnissen, jedoch oft komplex in der Auswertung und teurer in der Anschaffung \cite{Fraden:2016}.
	\item \textbf{Ultraschallsensoren:} Arbeiten nach dem Echo-Lot-Prinzip (Schalllaufzeitmessung). Sie senden Schallwellen aus und messen die Zeit, bis das Echo zurückkehrt \cite{SparkFun:2020}. Sie sind unabhängig von der Farbe oder optischen Transparenz eines Objekts, haben jedoch ihre Grenze bei schallabsorbierenden Materialien (z. B. Schaumstoff) \cite{Fraden:2016}.
\end{itemize}

\section{Kalibrierung}
Die Ultraschallmessung beruht auf der Ausbreitungsgeschwindigkeit von Schall in der Luft. Die Standardberechnung der Bibliothek geht von einer konstanten Schallgeschwindigkeit von $c \approx 343 \, \text{m/s}$ bei $20\,^\circ\text{C}$ aus \cite{Fraden:2016} . Da die Schallgeschwindigkeit temperaturabhängig ist, ändert sie sich um ca. $0,6 \, \text{m/s}$ pro Grad Celsius \cite{Fraden:2016}. Für hochpräzise Anwendungen muss das System kalibriert werden (z.B. über einen DHT22-Sensor). Die dynamische Berechnung lautet:
\[ \text{Distanz} = \frac{\text{Laufzeit}}{2} \times \left(331,3 + 0,606 \times \text{Temperatur in } ^\circ\text{C}\right) \]
