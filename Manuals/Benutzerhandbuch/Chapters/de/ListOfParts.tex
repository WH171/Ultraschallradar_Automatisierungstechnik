\chapter{Lieferumfang}
Obwohl das Gerät fertig montiert ist, sollten Sie prüfen, ob alle Systemkomponenten unbeschädigt vorhanden sind:

\begin{itemize}
	\item 1x Ultraschall-Radarsystem (fertig montierte Einheit auf Gehäuse)
	\item 1x USB-Verbindungskabel (USB-C auf USB-A)
	\item 1x Kurzanleitung (Quick-Start-Guide)
\end{itemize}
Sollten Teile fehlen oder Kabel lose sein, nehmen Sie das Gerät bitte nicht in Betrieb.

\section{Anforderungen an die Stromversorgung}
Für den reibungslosen Betrieb des Radarsystems ist eine stabile Energiequelle erforderlich. Sie können das System entweder direkt über einen USB-Port Ihres Laptops oder über ein externes USB-Netzteil betreiben.

\begin{itemize}
	\item \textbf{Anschlussart:} USB-C (Kabel im Lieferumfang enthalten).
	\item \textbf{Eingangsspannung:} $5\,V$ DC (Gleichspannung).
	\item \textbf{Empfohlene Stromstärke:} Mindestens $1\,A$ ($1000\,mA$).
\end{itemize}
\textit{Hinweis: Ein herkömmliches Smartphone-Ladegerät ist in der Regel vollkommen ausreichend.}
