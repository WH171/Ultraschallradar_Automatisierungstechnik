\chapter{Fehlersuche}

\begin{itemize}
	\item \textbf{Der Sensor dreht sich nicht:} Prüfen Sie, ob das USB-Kabel fest eingesteckt ist. Versuchen Sie einen anderen USB-Port an Ihrem Computer.
	\item \textbf{Kein Bild auf dem Monitor:} Stellen Sie sicher, dass in der Software der richtige „COM-Port“ ausgewählt wurde.
	\item \textbf{Falsche Entfernungen:} Prüfen Sie, ob der Sensor freie Sicht hat oder ob Kabel im Weg hängen, die selbst als Hindernis erkannt werden.
\end{itemize}
